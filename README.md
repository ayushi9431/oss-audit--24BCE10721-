## Project Description
This project is an audit of Python as an open-source software. It explores its origin, license, ecosystem, and practical usage on a Linux environment. It also includes shell scripts demonstrating basic Linux and scripting concepts.

---

## Scripts Overview

### Script 1: System Identity Report
Displays system information such as:
- Linux distribution
- Kernel version
- User and home directory
- System uptime and date

---

### Script 2: FOSS Package Inspector
- Checks if Python (or any package) is installed
- Displays version
- Uses case statement for description

---

### Script 3: Disk and Permission Auditor
- Checks important directories
- Displays permissions, owner, and size
- Uses loop and file checks

---

### Script 4: Log File Analyzer
- Reads a log file
- Counts occurrences of a keyword (default: error)
- Displays last matching lines

---

### Script 5: Open Source Manifesto Generator
- Takes user input
- Generates a personalized manifesto
- Saves output to a file
